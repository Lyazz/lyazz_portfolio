Cablin

Landing page for an Algerian low-voltage cable manufacturer, delivered as a fast Nuxt.js SSR website.

  • Role: Full Stack Engineer
  • Platform: Web (SSR)
  • Timeline: 2026

Overview

Cablin is a focused landing page for a low-voltage cable manufacturer in Algeria. The Nuxt.js build keeps the site performance-forward while presenting the product range and contact path in a clear, crawlable layout.

Key Features

  • Server-side rendering for quick first loads and search-friendly HTML output.
  • Reusable Vue.js sections to keep content blocks consistent and easy to extend.
  • Tailwind CSS utilities for responsive layout, spacing, and typography control.
  • SEO-ready metadata and structured page content for discoverability.

Tech Stack

  • Nuxt.js with SSR for the application framework.
  • Vue.js component architecture.
  • Tailwind CSS for styling.
