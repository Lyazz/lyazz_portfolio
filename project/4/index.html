QueueMaster

An IoT-enabled queue management ecosystem synchronizing mobile apps, desk displays, and custom hardware remotes.

  • Role: Lead IoT & Software Engineer
  • Platform: Flutter mobile, Android remote, BLE hardware
  • Timeline: 2020 — 2022

Overview

QueueMaster orchestrates patient and customer flow for hospitals and service centers. Flutter apps connect with BLE-powered remote controllers and wall displays, ensuring every station stays in sync. Administrators configure queue logic from the cloud while on-site staff manage calls with custom hardware that fits existing workflows.

Key Features

  • BLE remote controls with custom PCBs and 3D-printed housings for staff counters.
  • Real-time updates between displays, kiosks, and mobile devices via WebSockets.
  • Firebase Realtime Database stores queue states and analytics.
  • Multi-queue support with priority routing for critical departments.
  • Analytics dashboard highlighting peak hours and service-level performance.

Tech Stack

  • Flutter, Dart, and Provider for mobile clients.
  • Bluetooth Low Energy modules with Arduino firmware.
  • Firebase Realtime Database, Cloud Functions, and WebSockets.
  • 3D modeling in AutoCAD and 3D printing for enclosure prototyping.
  • Android SDK for dedicated remote control applications.
