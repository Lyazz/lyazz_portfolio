Excel Invoicing Tool

Desktop Flutter application that transforms raw Excel timesheets into curated activity catalogs and synthesis exports with a single click.

  • Role: Product Engineer
  • Platform: Windows & macOS Desktop
  • Timeline: 2023 — 2024

Overview

The Excel Invoicing Tool speeds up weekly invoice preparation by importing multiple spreadsheets, aggregating durations, and exporting a templated synthesis file. Team leads gain real-time visibility into hours, FTEs, and activity quality while keeping data safely stored on-device.

Key Features

  • Multi-file Excel import that aggregates per-row durations, captures employee metadata, and delivers contextual error feedback.
  • Guided onboarding with tutorial coach marks highlighting the import, weekly limits, and export actions for first-time users.
  • Responsive DataTable2 grids with filters, search, bulk selection, and batch deletion optimised for wide desktop layouts.
  • Smart weekly counters that flag capacity overages before synthesis and keep operators within template constraints.
  • Excel synthesis generator that outputs four weekly sheets, auto-calculates totals, and produces timestamped files.

Tech Stack

  • Flutter & Dart with AdaptiveTheme, DataTable2, TutorialCoachMark, AwesomeDialog, AnotherFlushbar, SharedPreferences, and Intl.
  • Excel parsing via the excel package with local persistence through sqflite_common_ffi and custom data models.
  • File management powered by file_picker for both import and export flows.

Operator Experience

Users can import timesheets across four weeks, see instant aggregation summaries, and curate activity catalogs before exporting. Inline warnings and confirmations protect data integrity, while contextual snackbars keep experienced users informed without interrupting flow.

Outcomes

  • Cuts manual consolidation time by auto-populating corporate synthesis templates with validated data.
  • Improves data quality through searchable history, inline alerts, and persistent local storage.
  • Provides polished onboarding and theme switching to reduce ramp-up time for new operators.
