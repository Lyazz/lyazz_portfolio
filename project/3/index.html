Stock Management App

An end-to-end logistics platform orchestrating production, warehousing, and distribution for a dairy manufacturer.

  • Role: Full-Stack Developer
  • Platform: Android, Web Dashboard
  • Timeline: 2021

Overview

The Stock Management App connects factory output with warehouse inventory and delivery fleets. Android field apps digitize stock intake and dispatch, while a responsive web dashboard gives managers real-time oversight of orders, routes, and invoicing. Automated PDF generation and barcode scans replaced paper-heavy workflows.

Key Features

  • Barcode and QR scanning for quick product identification and batch tracking.
  • GPS-enabled delivery routing with Google Maps API integration.
  • Role-based dashboards for production, logistics, and administration teams.
  • Automated PDF invoices, delivery notes, and quality audits.
  • REST API powering data sync between Android clients and the web portal.

Tech Stack

  • Java, Android SDK, Retrofit, and custom barcode modules.
  • CodeIgniter, PHP, and MySQL for backend services.
  • Responsive web dashboard using HTML, CSS, Bootstrap, and jQuery.
  • Google Maps Platform for fleet visualization and geofencing.
  • PDF generation, Excel export, and reporting pipelines.
